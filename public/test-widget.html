QR Code Widget Test Page

This page is for testing the embeddable QR code widget.

Scroll down to see the "auto show on scroll" feature in action (if enabled for the QR code).

Instructions

To test the widget, you need to replace YOUR_QR_CODE_ID in the script tag at the bottom of this file with a real QR code ID from your application.

  1. Go to your My QR Codes page.
  2. Find a QR code you want to test.
  3. Click the "Edit" button for that QR code.
  4. Copy the ID from the URL (e.g., in /create?id=..., the part after id= is the ID).
  5. Open public/test-widget.html in your code editor and paste the ID into the data-qr-id attribute of the script tag.
  6. Important: Make sure the QR code exists in your database and has show_button set to true.

Note: If nothing appears after updating the ID, check the browser console for errors. The most common issue is using an ID that doesn't exist in the database.
